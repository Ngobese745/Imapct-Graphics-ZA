# COOKIE AND TRACKING POLICY

**Last Updated: October 2, 2025**

## 1. INTRODUCTION

Impact Graphics ZA ("we," "our," or "us") uses cookies and similar tracking technologies in our mobile application to enhance your experience, analyze usage, and deliver personalized content.

This Cookie Policy explains what these technologies are, why we use them, and your choices regarding their use.

## 2. WHAT ARE COOKIES AND TRACKING TECHNOLOGIES?

### 2.1 Cookies
Small text files stored on your device that help us remember your preferences and activity.

**Note:** Traditional browser cookies are less common in native mobile apps, but similar technologies achieve the same purposes.

### 2.2 Mobile App Tracking Technologies

**Local Storage:**
- Data stored locally on your device
- Includes app preferences and settings
- Authentication tokens
- Cache data

**Device Identifiers:**
- Advertising IDs (IDFA on iOS, AAID on Android)
- Installation IDs
- Firebase Instance IDs

**SDKs (Software Development Kits):**
- Firebase Analytics
- Google AdMob
- Facebook SDK (for login)
- Google Sign-In SDK

**Session Tokens:**
- Authentication sessions
- API access tokens
- Temporary session data

## 3. TYPES OF TRACKING WE USE

### 3.1 Essential Tracking

**Purpose:** Necessary for app functionality  
**Cannot be disabled without affecting core features**

**What We Track:**
- User authentication status
- Session management
- Security tokens
- App configuration
- Error detection

**Technologies:**
- Firebase Authentication tokens
- Secure local storage
- Encrypted shared preferences

**Retention:**
- Session duration
- Until logout
- Until token expiry

### 3.2 Performance Tracking

**Purpose:** Monitor app performance and stability  
**Can be disabled:** Yes, in settings

**What We Track:**
- Crash reports
- Error logs
- Load times
- API response times
- Network performance
- Device compatibility

**Technologies:**
- Firebase Crashlytics
- Firebase Performance Monitoring
- Custom analytics events

**Data Collected:**
- Device model and OS version
- App version
- Screen resolution
- Network type
- Crash stack traces (no personal info)

**Retention:**
- 90 days for detailed logs
- 1 year for aggregated data

### 3.3 Analytics Tracking

**Purpose:** Understand user behavior and improve app  
**Can be disabled:** Yes, in settings

**What We Track:**
- Screens viewed
- Features used
- Time spent in app
- User flow patterns
- Button clicks
- Search queries (anonymized)
- Cart abandonment
- Conversion rates

**Technologies:**
- Firebase Analytics
- Google Analytics for Firebase
- Custom event tracking

**Data Collected:**
- Screen names
- Event names and parameters
- User properties (anonymized)
- Demographics (optional)
- Interests (inferred)

**Retention:**
- Raw data: 2 months
- Aggregated data: 14 months
- User-level data: 2 months

### 3.4 Advertising Tracking

**Purpose:** Deliver and measure advertisements  
**Can be disabled:** Yes, via device settings or app settings

**What We Track:**
- Ad views and clicks
- Ad conversions
- Advertising ID
- Ad interaction time
- Reward ad completions

**Technologies:**
- Google AdMob SDK
- Advertising identifiers (IDFA/AAID)
- Attribution tracking

**Data Collected:**
- Advertising ID
- Device information
- IP address (approximate location)
- Ad performance metrics

**Retention:**
- Per Google's AdMob policy
- Typically 90 days to 2 years

**Personalization:**
- Ads may be personalized based on:
  - In-app activity
  - Google ad profile
  - Interests inferred from usage
- Can opt-out of personalized ads

### 3.5 Functional Tracking

**Purpose:** Remember preferences and enhance experience  
**Can be disabled:** Partially

**What We Track:**
- Language preference
- Notification preferences
- Theme selection (dark/light mode)
- Sort and filter preferences
- Recently viewed items
- Search history

**Technologies:**
- Local shared preferences
- Secure storage
- App state management

**Retention:**
- Until changed by user
- Until app uninstall
- Duration of account

## 5. YOUR CHOICES AND CONTROLS

### 5.1 In-App Controls

**Settings > Privacy > Tracking Preferences:**

**Analytics:**
- Toggle: Enable/Disable analytics
- Effect: Stops usage tracking (essential tracking continues)
- Impact: None on app functionality

**Personalized Ads:**
- Toggle: Enable/Disable ad personalization
- Effect: Ads still shown but not personalized
- Impact: Less relevant ads

**Crash Reporting:**
- Toggle: Enable/Disable crash reports
- Effect: Stops sending crash data
- Impact: May affect support quality

**Performance Monitoring:**
- Toggle: Enable/Disable performance tracking
- Effect: Stops performance data collection
- Impact: None on user experience

### 5.2 Device-Level Controls

**Android:**
1. **Advertising ID Reset:**
   - Settings > Google > Ads > Reset advertising ID
2. **Opt-Out of Personalization:**
   - Settings > Google > Ads > Opt out of Ads Personalization
3. **Location:**
   - Settings > Apps > Impact Graphics ZA > Permissions > Location

**iOS:**
1. **Limit Ad Tracking:**
   - Settings > Privacy > Advertising > Limit Ad Tracking
2. **Reset Identifier:**
   - Settings > Privacy > Advertising > Reset Advertising Identifier
3. **Location:**
   - Settings > Privacy > Location Services > Impact Graphics ZA

### 5.3 Data Deletion

**Clear App Data:**
- Settings > Apps > Impact Graphics ZA > Storage > Clear Data
- Effect: Removes all local tracking data
- Impact: Must log in again, preferences reset

**Delete Account:**
- Settings > Account > Delete Account
- Effect: Removes all data associated with your account
- Impact: Permanent, cannot be undone
